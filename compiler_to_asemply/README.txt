--- starlet is a simplified c-like language ---
--- and starletc.py is the compiler from starlet to asemply---

to compile a stralet program, let's say test.stl,
type to your terminal "starletc.main(test.stl)"
then the followings file will automatically generated

1-> test.asm
    this file contains your (starlet) program in asemply conde

2-> test.c
    this file contains your (starlet) program in a much more simplified
    c code.

3-> test.int
    this is an intermediate stage of your (starlet) code to asemply one
